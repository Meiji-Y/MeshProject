# MeshProject
- You can create point cloud and mesh with using rgb-d camera. 
- After creation you can manage this mesh. 
- You can filter and cluster things in it. 
- You can find the object inside of other object.
## Requirments

- Python (3.10)
- OpenNI
- Open3D
- NumPy
- sk.learn
- matplotlib
